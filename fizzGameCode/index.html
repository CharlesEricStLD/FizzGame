Welcome to the fizz game

Here's the rule :

In this game the computer will ask you for a number to start with and the number of turn you want to do. After that entry, each player will have to enter the next number (if you enter 1, the next one is 2, and if you enter 17 the next one is 18 etc.), BUT, there is some special rules !

  1. If the number you enter is divisible by 3 should be replace by Fizz (so you write Fizz and not 3 at the prompt)
  2. If the number you enter is divisible by 5 should be replace by Buzz (so you write Buzz and not 3 at the prompt)
  3. The game will end after the number of turn you choose at the beguining of the game

Just push the start button down here to start the game!
